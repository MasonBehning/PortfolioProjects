					UP Paper Safety Management

This project aims to create a desktop application for managing safety incidents and reports at UP Paper.

						Installation

-Make sure you have an updated Python Installation.


-Run pip install -r requirements.txt

-Navigate to the "scripts" folder in your terminal.

-Run "python login_screen.py" to launch the application.

-Use the credentials "admin" for both the username and password to log in.


Note: Do not add date when generating report in the Form. Date will generate automatically. If date is added, it will break
